Portfolio



Calculator

This calculator application lets user perform basic mathematical operations. Its layout is similar to the one found on iPhones and MacBooks. As a bonus feature, I decided to create a button that would allow the user to change the color of the operator buttons. It takes in a number as an input and maps that number to a color. Whichever color matches the key becomeds the color of the buttons. If the user enters a number that is not found on the map, the color will change back to the default color.




Weather App

This Weather data site allows the user to enter a city and get basic weather data such as temperature, humidity, and wind speed. It recieves data from openweatherapi and maps it to the correct element. Depending on the weather description, the background will also change. For example, if the weather is rainy, the background will change to show rain. The same also holds true for snow, fog, and sunny weather.

Tic Tac Toe

A basic Tic Tac Toe game that can be played either with 2 people sharing a keyboard, or against the game's built in AI. The user can choose whether they would like to be an O or an X. The user will choose a space that has not yet been taken. In response, the AI will analyze the remaining spaces and choose based on which one is most likely to win the game. This continues until either one player gets three X's or O's in a row, or until the grid is filled. If neither player gets three in a row, the game will end in a draw.
